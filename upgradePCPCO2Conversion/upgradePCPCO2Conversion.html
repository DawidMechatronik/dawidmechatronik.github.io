External tank conversion (compressed air or CO2)

This is easy mod, you don't need any tools.

This is instruction for item sold by me on Etsy:

Click for more info

1. Disassembly

Remove CO2 cartridge pusher.

2. Assembly

Install external tank adapter

The end of adapter should fit nicely inside valve port

Hand tight the adapter and don't use too much force. In case of any leakage screw it a little more.

3. External tank connection

Now you need to decide what kind of external tank you want to connect.

3.1 External tank CO2 version

External tank with CO2, for example SodaStream. Click image for more info.

Softer trigger upgrade step by step instruction

Soft trigger mod with longer adjuster screws for first and second travel adjusting.

  • Factory trigger: 1st stage ~440g (1lbs), 2nd stage ~1300g (2,8lbs).

  • Soft spring mod: 1st stage ~200g (0,4lbs), 2nd stage ~600g (1,3lbs).

4. Trigger spring

You need to replace trigger return spring.

Now you need to install hammer catcher with new spring. Spring at both ends needs to in right positioning holes, it is very important!

5. Trigger disassembly and assembly

You need to replace adjustment bolts in trigger. After removing second travel distance adjuster screw some parts will fall out from trigger.

Here is correct order of those parts. Remember to put safety switch in correct direction!

You need to screw this little screw upside down. You can use pliers.

At the end you need to remove second travel distance adjuster screw.

6. Trigger one stage or two stage?

Now you need to decide which type of trigger you want to use. I think one stage is much better, because first travel adjusting screw is "locked" by spring. After setting trigger one, it stays in this setting. Second travel adjuster screw is without any locking mechanism (you can only add thread locking fluid). Without locking this screw will change trigger behaviour. This screw also likes to fall out.
